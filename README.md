# calculadora-uaeh

Calculadora de calificaciones desarrollada en C utilizando raylib, pensada para todos los estudiantes de la Universidad Autónoma del Estado de Hidalgo.

El proyecto nació con una idea simple: hacer una herramienta rápida, ligera y entretenida para calcular promedios y organizar calificaciones sin depender de aplicaciones pesadas o páginas saturadas, es por esto mismo que fue escrita enteramente en C.

Incluye una interfaz gráfica, y también algunos memes.

---

## Características

* Interfaz gráfica desarrollada con raylib
* Cálculo automático de promedios
* Sistema ligero y rápido
* Organización modular del código
* Compatible con Linux
* Estética inspirada en estudiantes universitarios

---

## Tecnologías utilizadas

* C
* raylib
* GCC
* Linux Mint

---

## Compilación

### Linux

Instalar raylib:

```bash
sudo apt install libraylib-dev
```

Compilar:

```bash
gcc *.c -o calculadora -lraylib -lm
```

Ejecutar:

```bash
./calculadora
```
